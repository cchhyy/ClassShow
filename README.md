班级活动管理系统/大学班级生活管理系统

1系统结构：Web端（服务器，客户端）+手机端（客户端）
2系统用户：管理员——添加用户
                       普通用户——浏览
		                              班级用户——可添加活动，评论
					                             班委、班主任、辅导员——添加活动，资料，审核
								     3主要功能：设计一个能够记录整个大学期间班级生活的系统，可分为班级活动（团日活动、专业学习，文体活动，公益活动），获奖展示，个人规划，班级事务，班级趣事，行业动态，共享资源，用户设置等功能。
								     添加、删除、查询、评论、上传下载
